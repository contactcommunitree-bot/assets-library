# Asset Library

Public asset depository. Browse a gallery, copy direct CDN URLs, paste into emails. Files served by jsDelivr (GitHub CDN); gallery hosted on GitHub Pages. Auto-rebuilt on every push.

**Live gallery:** https://contactcommunitree-bot.github.io/assets-library/

## How it works

```
assets/{organization}/{type}/{filename}.{ext}      ← the files
assets/{organization}/{type}/{filename}.tags.json  ← optional sidecar metadata
```

Each file's direct URL is:
```
https://cdn.jsdelivr.net/gh/contactcommunitree-bot/assets-library@main/assets/{org}/{type}/{filename}.{ext}
```

The gallery reads `site/assets.json` (regenerated by GitHub Actions on every push) and renders a filterable grid.

## Add new assets

1. Drop your file in the right folder: `assets/art4awareness/banner/earth-stories-hero.jpg`
2. (Optional) Add tags via sidecar:
   ```json
   { "campaigns": ["earth-stories-2026"], "alt": "Earth Stories 2026 hero banner" }
   ```
   Save as `assets/art4awareness/banner/earth-stories-hero.jpg.tags.json`
3. Commit + push:
   ```bash
   git add . && git commit -m "Add hero banner" && git push
   ```
4. GitHub Actions rebuilds the manifest and redeploys Pages (~1 min). jsDelivr serves the new file within ~10 min cache TTL.

If no sidecar is provided, the asset is tagged `campaigns: ["evergreen"]`.

## Taxonomy

Defined in `config.json`:

- **Organizations:** communitree, csr, ezone, art4awareness, save-a-turtle, fff-makers, littleleaders, tndwwt
- **Asset types:** logo, banner, header, footer, signature, icon, photo, poster, background, button, cta, divider
- **Campaigns:** evergreen, earth-stories-2026, weekly-newsletter, toy-ambassador, vermi-composter, pet-bottle-hero, e-waste-collector, unused-medicine, soil-healer (extend freely)

To extend, edit `config.json`. New organizations/types need matching folders under `assets/`.

## Local development

```bash
npm run build   # regenerate site/assets.json
npm run dev     # build + serve site/ on http://localhost:5173
```

## Architecture

- **Storage:** files committed to this public GitHub repo
- **CDN:** jsDelivr serves files at `cdn.jsdelivr.net/gh/...` (free, fast, global)
- **Gallery:** static HTML/CSS/JS in `site/` deployed to GitHub Pages
- **Build:** GitHub Actions workflow (`.github/workflows/deploy.yml`) runs the manifest script and deploys on every push to `main`
- **Cost:** $0/month — all on GitHub free tier

## File size & limits

- jsDelivr per-file limit: **50 MB** (images well under; for video use a different provider)
- GitHub repo soft limit: ~5 GB total
- For ~200 image assets: no concern

## Email integration

These URLs work in Gmail, Apple Mail, Outlook, and any major client:

```html
<img src="https://cdn.jsdelivr.net/gh/contactcommunitree-bot/assets-library@main/assets/art4awareness/logo/primary-2026.png"
     alt="Art for Awareness" style="width:200px; height:auto;">
```

Use the gallery's "Copy &lt;img&gt; tag" button to get a ready-made tag with `alt` and inline style.
